# mocha-chai-to-jest

How to migrate unit tests from mocha+chai to Jest.

## Getting started

Setup the project:
```
nvm use
npm install
```

### Run Mocha+chai tests

```
npm run test:mocha
```

### Run Jest tests

```
npm run test:jest
```

Bonus point if you want to see the code coverage with jest
```
npm run test:jest:coverage
```
